Seefood Website

COVID-19 Live Tracker India Map

This is a Live COVID-19 Tracker for India. Displays the number of active cases, confirmed cases, recovered cases and the death total for the country as well as for each state.

YelpCamp

YelpCamp is a website where users can create and review campgrounds. In order to review or create a campground, you must have an account. This project was part of Colt Steele's web dev course on udemy. This project was created using Node.js, Express, MongoDB, and Bootstrap. Passport.js was used to handle authentication.

Pokedex React App

It is a simple Pokemon Game. The App renders a Pokegame component which picks random Pokemons and assigns them in two groups. It then calculates the total experience of each group. The group with more experience is the winner
